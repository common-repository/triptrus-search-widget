Plugin for displaying TripTrus Search Widget

== Description ==
Plugin for displaying TripTrus Search Widget in sidebar

== Installation ==
1. Upload "triptrus-search-widget" directory to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= What is Operator ID? =
Operator ID is your triptrus.com profile id.  
You can get your id by visiting [this site](http://www.triptrus.com/your-profile-id)

= What is the purpose of Operator ID in search widget? =
Widget will search trips in this operator's trip only
